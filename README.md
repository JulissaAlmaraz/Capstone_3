# Capstone_3
Capstone 3 EmporiUm Sales Analysis

This project analyzes EmporiUm sales data from 2022–2025 using Power BI. The report was created for the West Regional Director to identify sales trends, category performance, state sales distribution, and top-selling general audience books within the West region.

Tools Used
Power BI
Power Query
DAX
Project Includes
Sales trend analysis
Sales by category
Sales by state
West region sales analysis
Top-selling books report (excluding textbooks)
